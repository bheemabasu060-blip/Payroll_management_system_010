Payroll Management System using MySQL (CMD), PHP Frontend, and GitHub Deployment

Project Description

The Payroll Management System is a comprehensive database-driven web application developed using MySQL Command Line Interface (CLI) for backend database management and PHP, HTML5, CSS3, and JavaScript for frontend development. The system is designed to automate and streamline payroll processing, employee management, attendance tracking, salary administration, leave management, and payment processing within an organization.

The project demonstrates the practical implementation of Database Management System (DBMS) concepts including relational database design, normalization, SQL query processing, stored procedures, triggers, constraints, joins, subqueries, and transaction management. The application follows a structured client-server architecture where the frontend interacts with the MySQL database through PHP using PDO/MySQLi connectivity.

The database schema consists of seven normalized entities: Employee, Salary, Attendance, Payroll, Department, Payment, and Leave_Request. These entities are linked through primary and foreign key relationships to ensure referential integrity and efficient data management. The system supports complete CRUD (Create, Read, Update, Delete) operations for employee records, salary details, attendance information, payroll generation, and payment transactions.

Advanced SQL programming techniques are implemented through stored procedures that automate payroll calculation and salary statement generation. Trigger mechanisms are used to automatically generate payroll records after attendance updates and prevent duplicate payroll generation for the same employee within a payroll cycle. Various SQL operations including INNER JOIN, LEFT JOIN, GROUP BY, HAVING, correlated subqueries, and nested subqueries are executed and validated through MySQL CLI.

The frontend interface provides modules for employee registration, attendance management, salary processing, payroll generation, leave request handling, payment tracking, and report generation. Users can efficiently manage employee data, update salary structures, monitor attendance records, generate payroll reports, and view payment histories through an interactive web interface.

Version control and project deployment are managed using Git and GitHub, ensuring proper source code management, collaboration, documentation, and project maintenance. The repository contains SQL scripts, frontend source code, database connection modules, screenshots, project documentation, and deployment resources.

Project Objective

To design, develop, implement, and demonstrate a fully functional Payroll Management System that integrates relational database management, procedural SQL programming, trigger automation, frontend-backend communication, and version-controlled deployment. The system aims to automate payroll operations, improve data accuracy, ensure database integrity, reduce manual processing effort, and provide an efficient platform for employee payroll administration.
